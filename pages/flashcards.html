Wie das Leitner-System funktioniert
Each card is in a box (1–5). Gewusst moves it one box up (longer interval). Nochmal sends it back to box 1. Intervals: Box 1 = same day, 2 = 1 day, 3 = 3 days, 4 = 7 days, 5 = 16 days. Reach Box 5 = learned. All progress is saved on this device only.
